Create venv
===========

.. code-block:: bash

   python3 -m venv venv

Activate venv
=============

.. code-block:: bash

   . venv/bin/activate

Install Dependencies
====================

.. code-block:: bash

   pip install -r requirements.txt

Initialize Database
===================

.. code-block:: bash

   python app.py --init-db

Run Web Server Locally
======================

.. code-block:: bash

   python app.py

==========

API documentation is available at ``/static/swagger-ui.html``.
